## About Myself 👋
My name is Musa Demir.

I graduated from the Computer Engineering department at Sakarya University.

I worked more than 2.5 years in the Sakarya University IT department, where we developed and marketed university management systems to numerous universities.

Subsequently, I began working as one of the main developers for  [AspNet Boilerplate](https://github.com/aspnetboilerplate/aspnetboilerplate) and [AspNet Zero](https://aspnetzero.com/) at Volosoft.

After a fulfilling journey of developing web frameworks for over 3.5 years at Volosoft, I transitioned to BigPogo Games as a Full-Stack Backend Developer.

Currently, at BigPogo Games, I handle server-side requirements, web APIs, etc., for a multiplayer casual mobile game, and also implement game logic.

My primary programming language is C#. Since the beginning of my professional career, I've been working as a Full-Stack developer. This has given me the opportunity to professionally develop using Javascript, Typescript, Angular, Unity and a lot of other frameworks and libraries.

I've been developing web applications and frameworks professionally for over 6 years.

This is a brief summary of my software journey so far.
